General¶

git --version

git help config
git help -a # all
git help -g # guide

git config --list

git config --global user.name "Name Surname"
git config --global user.email "name.surname@gmail.com"
git config --global code.editor nano

git config --system user.name
git config --global user.name
git config --local user.name

# Create repository in existing directory.
git init

# Clone an existing repository.
git clone https://aaa@bitbucket.org/aaa/git-tutorial.git

Status¶

git status

git status --short

?? new file created

A_ new file created and staged

AM new file created, staged and then modified

M_ modified and staged

MM modified, staged and then modified

_M modified and not staged

_D file deleted

D_ file deleted and staged

R_ file renamed and staged

UU` |arrow-right| merge conflict

Diff¶

git diff for _M

git diff --staged for M_

Commit¶

git commit -m "Commit message"

git commit -a -m "Commit message" for skipping staging area

Remove¶

git rm file_name removed and staged

git rm --cached file_name removed and staged, also untracked

git rm -f file_name removed and staged (modified files)

Rename¶

git mv file_name new_file_name

Log¶

git log

git log -2 last two entries

git log -p with diff

git log --stat with diff

git log --pretty=oneline

git log --pretty=format:"%h %an %ar %s"

git log --pretty=format:"%h %s" --graph

git log --since=1.hour

Undoing things¶

git commit --amend change commit message or add changes to the last commit

Warning

Do not do this after git push

git reset HEAD file_name unstage changes

git checkout -- file_name remove all changes from working directory

Warning

Not recoverable operation.

Remote¶

git remote show remotes

git remote -v show remotes and URL’s

git remote add remote_name remote_url add remote

git remote show origin show more info

git remote rename old_remote_name new_remote_name rename remote

git remote rm remote_name

Fetch¶

git fetch remote_name

Pull¶

git pull fetch and merge

Push¶

git push origin master

Tag¶

git tag show tags

git tag 0.0.1 create lightweight

git tag -a -m "" 0.0.1 create annotated tag

git show 0.0.1 show tagged commit

git tag -d 0.0.1 delete tag

git tag -a -m "" 0.0.1 commit_checksum create annotated (lightweight) tag later

git push origin 0.0.1 push one tag

git push origin --tags push all tags

Alias¶

git config --global alias.st "status --short"
